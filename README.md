# Next-Generation-Interactive-and-Immersive-Graphs
A Unity-Based Framework for Advanced Data Visualization.

This Master's thesis was focused on the development of an immersive data visualization framework focused on social network graphs, incorporating exploration and interactive visualization techniques that enhance user engagement and comprehension of the depicted data. This framework is available on Desktop and VR devices.
